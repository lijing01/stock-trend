| 7 | **入场时机细化** | 放量突破确认（量 > 1.3x MA20 量）、RSI 拐点入场、布林带收窄后突破、缩量回踩支撑 | `etf_scanner.py:529-622` |

## 实施顺序建议

按依赖关系排列：

1. **P0 #1** → 任何优化前提（评分本身有偏，先修基线）
2. **P1 #2** → 回测框架完善后，后续改动才有量化评估依据
3. **P1 #3 + #4** → 大环境判断 + 下跌惩罚，与 #2 无依赖，可并行
4. **P2 #5 → #8 → #6 → #7 → #9** → 止损是风控底线优先，仓位公式其次，板块轮动和入场细化再次
5. **P3 #10-#14** → 基础设施类，依赖前期改动稳定后再接入
