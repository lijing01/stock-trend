# K线形态识别参考

## 1. 单根K线特征计算

对每根K线计算以下特征值，用于形态识别：

| 特征 | 计算公式 | 说明 |
|---|---|---|
| 实体大小 | `abs(close - open)` | 阳线实体 = close - open，阴线实体 = open - close |
| 上影线 | `high - max(open, close)` | 最高价到实体上边界的距离 |
| 下影线 | `min(open, close) - low` | 实体下边界到最低价的距离 |
| 全幅 | `high - low` | 最高价与最低价之差 |
| 方向 | `close > open` 为阳线，`close < open` 为阴线 | 阳线=看涨，阴线=看跌 |
| 涨跌幅 | `(close - pre_close) / pre_close` | 与前日收盘价对比 |

**判定阈值：**

| 特征 | 条件 | 判定 |
|---|---|---|
| 十字星 | `abs(close - open) <= 0.05 * (high - low)` | 开收盘几乎相等 |
| 小实体 | `abs(close - open) < 0.2 * (high - low)` | 实体占全幅比例小 |
| 大实体 | `abs(close - open) > 0.6 * (high - low)` | 实体占全幅比例大 |
| 长上影 | `上影线 >= 2 * 实体` | 上方抛压大 |
| 长下影 | `下影线 >= 2 * 实体` | 下方支撑强 |
| 无影线 | `(上影线 + 下影线) < 0.1 * 实体` | 光头光脚 |

## 2. 看多反转形态

### 2.1 锤子线 (Hammer)

| 属性 | 说明 |
|---|---|
| 识别条件 | 下影线 >= 2x 实体，上影线 <= 0.3x 实体，实体为阴或阳均可 |
| 前提条件 | 出现在下跌趋势中（MA5 < MA20 或近5日多数为阴线） |
| 得分 | +1 ~ +2（放量锤子线得分偏高） |

### 2.2 倒锤子 (Inverted Hammer)

| 属性 | 说明 |
|---|---|
| 识别条件 | 上影线 >= 2x 实体，下影线 <= 0.3x 实体，实体为阳线 |
| 前提条件 | 出现在下跌趋势中 |
| 得分 | +1（需次日阳线确认） |

### 2.3 早晨之星 (Morning Star)

| 属性 | 说明 |
|---|---|
| 识别条件 | (1) 大阴线 (2) 十字星或小实体，与(1)有缺口 (3) 大阳线，收盘深入(1)实体50%+ |
| 前提条件 | 出现在下跌趋势末期 |
| 得分 | +2 ~ +3（三根K线组合，信号最强） |

### 2.4 看涨吞没 (Bullish Engulfing)

| 属性 | 说明 |
|---|---|
| 识别条件 | (1) 阴线 (2) 阳线，(2)实体完全包含(1)实体 |
| 前提条件 | 出现在下跌趋势中 |
| 得分 | +2 |

### 2.5 刺透形态 (Piercing Line)

| 属性 | 说明 |
|---|---|
| 识别条件 | (1) 阴线 (2) 阳线，(2)开盘低于(1)最低价，收盘超过(1)实体中点 |
| 前提条件 | 出现在下跌趋势中 |
| 得分 | +1 ~ +2 |

### 2.6 三白兵 (Three White Soldiers)

| 属性 | 说明 |
|---|---|
| 识别条件 | 连续3根阳线，每根开盘在前根实体内，收盘创新高 |
| 前提条件 | 下跌趋势反转 |
| 得分 | +2 ~ +3 |

### 2.7 跳空向上 (Bullish Gap Up)

| 属性 | 说明 |
|---|---|
| 识别条件 | 当日最低价 > 前日最高价 |
| 前提条件 | 下跌反转或突破确认 |
| 得分 | +1 ~ +2 |

## 4. 中性/犹豫形态

### 4.1 十字星 (Doji)

| 属性 | 说明 |
|---|---|
| 识别条件 | `abs(close - open) <= 0.05 * (high - low)` |
| 含义 | 趋势末端出现 = 可能反转；趋势中段 = 犹豫 |
| 得分 | 0（方向取决于出现位置） |

### 4.2 纺锤线 (Spinning Top)

| 属性 | 说明 |
|---|---|
| 识别条件 | 实体 < 全幅 * 0.2，上下影线均较长 |
| 含义 | 多空犹豫，趋势不确定 |
| 得分 | 0 |

## 5. 形态识别流程

1. **判断当前趋势方向**：使用 MA5 与 MA20 的关系，或最近5根K线的涨跌数量
2. **扫描最近10根K线**，按优先级检测：
   - 优先：3根K线组合形态（早晨之星、黄昏之星、三白兵、三只乌鸦）
   - 其次：2根K线组合形态（看涨/看跌吞没、刺透形态、乌云盖顶）
   - 最后：单根K线形态（锤子线、射击之星、十字星等）
3. **验证前提条件**：形态必须出现在对应趋势位置才有效（下跌末期的锤子线有效，上涨末期的锤子线无效）
4. **合并同向信号**：多个同向形态得分可叠加，但技术面维度总分不超过 ±3
5. **冲突处理**：方向冲突时，组合形态优先于单根形态，降低单根形态权重

## 6. 数据要求

- 最少需要 **10根K线** 数据才能执行形态分析
- 需要至少 **60根K线** 才能计算 MA60 均线
- 数据不足时仅分析可用部分，报告中标注数据限制
- 前复权数据（`adj='qfq'`）用于形态识别更准确
- 港股不支持复权参数，使用原始价格数据
