# 趋势判断维度 — 详细检查标准

## 1. 技术面 (Technical)

### 1.1 MA 均线

**检查项**: MA5 / MA10 / MA20 / MA60

| 信号 | 条件 | 得分 |
|---|---|---|
| 多头排列 | MA5 > MA10 > MA20 > MA60 | +2 ~ +3 |
| 空头排列 | MA5 < MA10 < MA20 < MA60 | -2 ~ -3 |
| 金叉 | 短期均线上穿长期均线 | +1 |
| 死叉 | 短期均线下穿长期均线 | -1 |
| 缠绕 | 均线密集，无明确方向 | 0 |

### 1.2 MACD

**检查项**: DIF / DEA / 柱状体

| 信号 | 条件 | 得分 |
|---|---|---|
| 金叉+红柱放大 | DIF 上穿 DEA，红柱持续增长 | +2 |
| 金叉+红柱缩窄 | DIF 上穿 DEA，红柱开始缩短 | +1 |
| 死叉+绿柱放大 | DIF 下穿 DEA，绿柱持续增长 | -2 |
| 顶背离 | 价格创新高，MACD 不创新高 | -2 ~ -3 |
| 底背离 | 价格创新低，MACD 不创新低 | +2 ~ +3 |

### 1.3 RSI

**检查项**: 14日 RSI

| 信号 | 条件 | 得分 |
|---|---|---|
| 超买 | RSI > 70 | -1 ~ -2 |
| 超卖 | RSI < 30 | +1 ~ +2 |
| 中性 | 30 ≤ RSI ≤ 70 | 0 |
| 背离 | 价格与 RSI 方向相反 | ±2 |

### 1.4 KDJ

**检查项**: K / D / J 三线

| 信号 | 条件 | 得分 |
|---|---|---|
| 低位金叉 | K 线在 20 以下上穿 D 线 | +1 ~ +2 |
| 高位死叉 | K 线在 80 以上下穿 D 线 | -1 ~ -2 |
| 超买区 | K > 80 | -1 |
| 超卖区 | K < 20 | +1 |

### 1.5 布林带

**检查项**: 上轨 / 中轨 / 下轨

| 信号 | 条件 | 得分 |
|---|---|---|
| 突破上轨 + 放量 | 价格突破上轨且成交量放大 | +2 |
| 跌破下轨 | 价格跌破下轨 | -1 ~ -2 |
| 收口 | 布林带宽度缩窄，变盘前兆 | 0（注意方向） |
| 中轨支撑 | 价格在中轨上方运行 | +1 |

### 1.6 成交量

**检查项**: 量价配合

| 信号 | 条件 | 得分 |
|---|---|---|
| 放量上涨 | 量增价升 | +2 |
| 缩量下跌 | 量缩价跌 | -1（趋势弱化） |
| 天量天价 | 成交量创历史新高+价格高位 | -2（见顶信号） |
| 地量地价 | 成交量极低+价格低位 | +1（见底可能） |

### 2.4 龙虎榜

## 3. 基本面 (Fundamental)

### 3.1 估值 (PE/PB)

| 信号 | 条件 | 得分 |
|---|---|---|
| 低估 | PE/PB 处于历史 20% 分位以下 | +2 |
| 合理 | PE/PB 处于历史 20%-80% 分位 | 0 |
| 高估 | PE/PB 处于历史 80% 分位以上 | -2 |
| 行业对比 | 低于行业中位数 | +1 |

### 3.2 业绩增速

| 信号 | 条件 | 得分 |
|---|---|---|
| 加速增长 | 营收/利润增速连续加快 | +2 |
| 稳定增长 | 营收/利润稳定正增长 | +1 |
| 下滑 | 增速放缓或负增长 | -1 ~ -2 |
| 亏损 | 净利润为负 | -2 ~ -3 |

### 3.3 行业景气度

| 信号 | 条件 | 得分 |
|---|---|---|
| 景气上行 | 行业 PMI/景气指数上行 | +1 |
| 政策扶持 | 产业政策明确支持 | +1 |
| 景气下行 | 行业景气下滑 | -1 |
| 政策打压 | 监管收紧 | -2 |

### 3.4 分红股息率

| 信号 | 条件 | 得分 |
|---|---|---|
| 高股息 | 股息率 > 10 年国债收益率 + 2% | +1 |
| 低股息 | 股息率 < 1% | 0（成长股中性） |

## 4. 情绪面 (Sentiment)

### 4.1 涨停/跌停数

| 信号 | 条件 | 得分 |
|---|---|---|
| 涨停潮 | 涨停数 > 50 | +2 |
| 跌停潮 | 跌停数 > 50 | -2 |
| 涨多跌少 | 上涨家数远多于下跌 | +1 |
| 跌多涨少 | 下跌家数远多于上涨 | -1 |

### 4.2 换手率

| 信号 | 条件 | 得分 |
|---|---|---|
| 温和放量 | 换手率 3%-8% 且价格上涨 | +1 |
| 暴量 | 换手率 > 15% | -1（短期过热） |
| 极度萎缩 | 换手率 < 1% | 0（观望） |

### 4.3 板块联动

| 信号 | 条件 | 得分 |
|---|---|---|
| 强联动 | 同板块 80%+ 个股同向运动 | +1（趋势确认） |
| 分化 | 板块内个股方向不一致 | -1（趋势不牢） |

### 4.4 新闻舆情

| 信号 | 条件 | 得分 |
|---|---|---|
| 利好集中 | 多条正面新闻 | +1 |
| 利空集中 | 多条负面新闻 | -1 |
| 静默期 | 无重大新闻 | 0 |

## 港股特殊检查项

| 检查项 | 说明 |
|---|---|
| 恒生指数联动 | 港股与恒指走势高度相关，需单独分析恒指趋势 |
| 卖空占比 | 卖空成交额占总成交比例，>15% 偏空 |
| 南向资金 | 内地资金流入港股情况 |
| AH 溢价 | AH 股溢价指数，溢价过高偏空 |
| 无涨跌停 | 需额外提示波动风险 |

## ETF 特殊检查项

| 检查项 | 说明 |
|---|---|
| IOPV 折溢价 | 实时净值与交易价格偏差，溢价过高偏空 |
| 跟踪误差 | 与基准指数的偏离度 |
| 申赎清单 | 一级市场申赎状态影响流动性 |
| 成交额 | ETF 流动性指标，过低则趋势信号弱 |
