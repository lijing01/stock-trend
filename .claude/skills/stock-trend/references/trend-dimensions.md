# 趋势判断维度 — 评分标准

## 1. 技术面 (Technical) [-3, +3]

### 1.1 MA 均线

| 信号 | 条件 | 得分 |
|---|---|---|
| 多头排列 | MA5 > MA10 > MA20 > MA60 | +2 ~ +3 |
| 空头排列 | MA5 < MA10 < MA20 < MA60 | -2 ~ -3 |
| 金叉/死叉 | 短期均线上穿/下穿长期均线 | ±1 |
| 缠绕 | 均线密集，无明确方向 | 0 |

### 1.2 MACD

| 信号 | 条件 | 得分 |
|---|---|---|
| 金叉+红柱放大 | DIF 上穿 DEA，红柱增长 | +2 |
| 金叉+红柱缩窄 | DIF 上穿 DEA，红柱缩短 | +1 |
| 死叉+绿柱放大 | DIF 下穿 DEA，绿柱增长 | -2 |
| 顶/底背离 | 价格与 MACD 方向相反 | ±2~3 |

### 1.3 RSI (14日)

| 信号 | 条件 | 得分 |
|---|---|---|
| 超买 | RSI > 70 | -1 ~ -2 |
| 超卖 | RSI < 30 | +1 ~ +2 |
| 中性 | 30 ≤ RSI ≤ 70 | 0 |

### 1.4 KDJ

| 信号 | 条件 | 得分 |
|---|---|---|
| 低位金叉 | K 在 20 以下上穿 D | +1 ~ +2 |
| 高位死叉 | K 在 80 以上下穿 D | -1 ~ -2 |

### 1.5 布林带

| 信号 | 条件 | 得分 |
|---|---|---|
| 突破上轨+放量 | 价格突破上轨且量放大 | +2 |
| 跌破下轨 | 价格跌破下轨 | -1 ~ -2 |
| 收口 | 布林带宽度缩窄 | 0（注意方向） |
| 中轨支撑 | 价格在中轨上方 | +1 |

### 1.6 成交量

| 信号 | 条件 | 得分 |
|---|---|---|
| 放量上涨 | 量增价升 | +2 |
| 缩量下跌 | 量缩价跌 | -1 |
| 天量天价 | 量创新高+价格高位 | -2 |
| 地量地价 | 量极低+价格低位 | +1 |

### 1.7 K线形态

详见 **[kline-patterns.md](kline-patterns.md)**。使用 `analyze_technical.py` 的 `patterns` 输出判断。

### 1.8 ADX（趋势强度）

| 信号 | 条件 | 得分 |
|---|---|---|
| 强趋势 | ADX > 40 | ±1（方向由DI确认） |
| 趋势有效 | 25 < ADX ≤ 40 | ±1 |
| 趋势弱 | 20 < ADX ≤ 25 | 0 |
| 震荡市 | ADX ≤ 20 | 0（避免追涨杀跌） |

+DI > -DI 且 ADX > 25 → 多头确认(+1)；-DI > +DI 且 ADX > 25 → 空头确认(-1)

### 1.9 OBV（能量潮）

| 信号 | 条件 | 得分 |
|---|---|---|
| OBV上穿20日均线 | 资金持续净流入 | +1 |
| OBV下穿20日均线 | 资金持续净流出 | -1 |
| OBV底背离 | 价格新低但OBV未新低 | +1（量先价行） |
| OBV顶背离 | 价格新高但OBV未新高 | -1（量先价行） |

### 1.10 ATR（波动率）

ATR 不直接参与评分，但影响**仓位建议**和**止损位计算**：

| ATR% (ATR/价格) | 波动率等级 | 仓位建议 |
|---|---|---|
| > 4% | 极端波动 | 轻仓(20-30%) |
| 2.5% ~ 4% | 高波动 | 半仓(40-50%) |
| 1.5% ~ 2.5% | 正常波动 | 标准仓位(50-70%) |
| < 1.5% | 低波动 | 可重仓(70-80%) |

止损位 = max(支撑位 - 1×ATR, 当前价 - 2×ATR)

### 技术面子权重

`analyze_technical.py` 的 `build_summary` 自动应用以下子权重：

| 类别 | 指标 | 子权重 |
|---|---|---|
| 趋势指标 | MA、MACD | ×1.5 |
| 趋势强度 | ADX | ×1.2 |
| 震荡指标 | RSI、KDJ | ×0.8 |
| 通道/量能 | 布林带、成交量、OBV | ×1.0 |
| K线形态 | patterns | ×0.5 |

**一致性因子**：同向指标数/总指标数影响置信度。5个全偏多 vs 3偏多2偏空，即使加权总分相同，前者置信度更高。

得分合并规则：加权平均后四舍五入至 [-3, +3]；冲突时组合形态优先于单根形态。

## 6. 单事件封顶规则

单一事件对维度得分贡献有上限：

| 维度 | 单事件封顶 | 高分最低信号数 |
|---|---|---|
| 宏观面 | 1.5 | 2（得分>1.5需≥2条独立信号） |
| 资金面 | 1.0 | 2（得分>1.0需≥2条独立信号） |
| 基本面 | 1.0 | 2（得分>1.0需≥2条独立信号） |
| 情绪面 | 1.0 | 2（得分>1.0需≥2条独立信号） |

**判定标准**：同一天同一主题的新闻算一个事件，不重复计分。

## 7. 正反强制规则

每个非技术维度必须同时列出至少1条利好+1条利空。

**摘要格式**：
```
资金面：利多：ETF近20日净申购+1.75亿；利空：主力5/12净流出1.54亿
```

**违反处理**：
- 搜索结果只有单向信号 → 标注"未找到反向信号，可能存在确认偏差"
- 缺少反向信号 → 该维度一致性因子×0.6

**反向验证搜索词**（追加到Step 3搜索关键词后）：

| 维度 | 反向验证词 |
|---|---|
| 资金面 | 流出 危机 减持 |
| 基本面 | 风险 下滑 亏损 |
| 情绪面 | 下跌 跌停 恐慌 |
| 宏观面 | 鹰派 衰退 收紧 |

## 港股特殊检查项

恒指联动、卖空占比(>15%偏空)、南向资金、AH溢价(过高偏空)、无涨跌停提示

## ETF 特殊检查项

IOPV折溢价(过高偏空)、跟踪误差、申赎状态、成交额(过低信号弱)

## 可转债特殊检查项

转股溢价率(过高偏空)、纯债价值(接近=偏多有底)、强赎风险(接近触发价=偏空)

## 9. ETF期货信号维度归属（仅ETF标的）

数据源：`futures_data.json`，由 `fetch_futures_data.py` 自动获取。

### 期现价差(基差)评分标准

| 基差百分比 | 方向 | 信号 | 得分 |
|---|---|---|---|
| > 2% | 大幅升水 | 看多，但注意过度投机可能反转 | +1 |
| 0.5% ~ 2% | 升水 | 轻度看多 | +0.5 |
| -0.5% ~ 0.5% | 平水 | 中性 | 0 |
| -2% ~ -0.5% | 贴水 | 轻度看空 | -0.5 |
| < -2% | 大幅贴水 | 看空，但注意过度恐慌可能反转 | -1 |

### 持仓量趋势(OI)评分标准

| 价格方向 | OI变化 | 信号 | 得分 |
|---|---|---|---|
| 上涨 | OI增加 | 强看多（新多头入场） | +1 |
| 下跌 | OI增加 | 强看空（新空头入场） | -1 |
| 上涨 | OI减少 | 弱反弹（空头回补） | -0.5 |
| 下跌 | OI减少 | 弱下跌（多头平仓） | +0.5 |

### 期货成交量确认评分标准

| 成交量 vs 5日均量 | 信号 | 得分 |
|---|---|---|
| > 120% 且趋势一致 | 确认趋势 | +0.5 |
| < 80% | 趋势可疑 | -0.5 |
| 其他 | 中性 | 0 |

### 数据不可用处理

| 期货品种 | 数据源 | 降级方案 |
|---|---|---|
| A股指数期货(IF/IH/IC/IM) | 东方财富API → AKShare | 静默跳过，不影响其他维度 |
| 港股指数期货(HTI_M/HSI_M) | 东方财富API | 境外网络不可达时返回 `unavailable`，静默跳过 |
| 无映射的ETF | — | 返回 `unsupported`，不获取期货数据 |
