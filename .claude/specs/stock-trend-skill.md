# 股票趋势判断 Skill — 功能规格说明 v1

## 1. 概览

| 项目 | 说明 |
|---|---|
| 名称 | `stock-trend` |
| 触发方式 | `/stock-trend` 或 `/stock-trend 600519` |
| 用途 | 对 A股、港股、ETF 输出当日趋势判断（看多/看空/震荡）及依据摘要 |
| 输出 | 结构化趋势报告 |

## 2. 输入规范

```
/stock-trend <code> [options]

code:   股票/ETF代码，自动识别市场
        - 6位纯数字 → A股 (000001=平安银行, 510050=上证50ETF)
        - 5位纯数字 → 港股 (00700=腾讯)
        - 带后缀   → 显式指定 (600519.SH, 00700.HK)

options:
  --focus <维度>    侧重维度: technical | capital_flow | fundamental | sentiment
  --horizon <周期>   判断周期: intraday | daily | weekly (默认 daily)
  --compact         精简模式，只输出结论
```

### 输入验证规则

1. `code` 为必填参数，缺失时提示用法
2. 代码格式不匹配任何市场规则时，报错并给出示例
3. `--focus` 可叠加使用，如 `--focus technical --focus capital_flow`
4. `--horizon` 仅接受 `intraday | daily | weekly`，其他值报错

## 3. 市场与标的识别

| 代码模式 | 市场 | 示例 |
|---|---|---|
| `6xxxxx` / `6xxxxx.SH` | 上交所 A股 | 600519.SH 贵州茅台 |
| `0xxxxx` / `0xxxxx.SZ` | 深交所 A股 | 000001.SZ 平安银行 |
| `3xxxxx` / `3xxxxx.SZ` | 创业板 | 300750.SZ 宁德时代 |
| `688xxx` / `688xxx.SH` | 科创板 | 688981.SH 中芯国际 |
| `5xxxxx` | 上交所 ETF/基金 | 510050 上证50ETF |
| `15xxxx` | 深交所 ETF | 159919 沪深300ETF |
| `0xxxx` / `0xxxx.HK` | 港股 | 00700 腾讯控股 |
| `0xxxx.HK` | 港股 ETF | 02800 盈富基金 |

### 标的类型附加标记

- A股代码前标 `ST` 或 `*ST` → 标记退市风险警示
- 港股代码 → 标记无涨跌停限制
- ETF 代码 → 标记需分析 IOPV 折溢价

## 4. 分析维度（五维模型）

### 4.1 技术面 (Technical)

| 指标 | 说明 | 看多信号 | 看空信号 |
|---|---|---|---|
| MA 均线 | MA5/MA10/MA20/MA60 | 多头排列 | 死叉 |
| MACD | DIF/DEA/柱状体 | 金叉+红柱放大 | 顶背离 |
| RSI | 14日RSI | <30 超卖反弹 | >70 超买回落 |
| KDJ | K/D/J 三线 | 低位金叉 | 高位死叉 |
| 布林带 | 上轨/中轨/下轨 | 突破上轨+放量 | 跌破下轨 |
| 成交量 | 量价配合 | 放量上涨确认 | 缩量下跌 |

### 4.2 资金面 (Capital Flow)

| 指标 | 说明 | 判断依据 |
|---|---|---|
| 主力净流入 | 大单/中单/小单资金流向 | 大单持续净流入=看多 |
| 北向资金 | 沪股通/深股通净买入 | 连续净买入=看多（仅A股） |
| 融资融券 | 融资余额变化趋势 | 余额持续上升=看多 |
| 龙虎榜 | 机构/游资席位动向 | 机构净买入=看多 |

### 4.3 基本面 (Fundamental)

| 指标 | 说明 | 判断依据 |
|---|---|---|
| PE/PB | 估值百分位（历史分位） | 低于中位数=偏低估 |
| 业绩增速 | 营收/净利润同比增长 | 加速增长=看多 |
| 行业景气度 | 所属行业景气指数/政策 | 景气上行=看多 |
| 分红股息率 | 股息率与无风险利率 | 股息率>国债收益率=偏多 |

### 4.4 市场情绪 (Sentiment)

| 指标 | 说明 | 判断依据 |
|---|---|---|
| 涨停/跌停数 | 市场赚钱效应 | 涨停>>跌停=情绪偏多 |
| 换手率 | 活跃度指标 | 温和放量=健康 |
| 板块联动 | 同板块个股一致性 | 高度联动=趋势确认 |
| 新闻舆情 | 利好/利空消息密度 | 利好集中=短期偏多 |

### 4.5 宏观/政策 (Macro)

| 因素 | 说明 | 判断依据 |
|---|---|---|
| 货币政策 | 降准/降息/流动性 | 宽松=偏多 |
| 行业政策 | 监管文件/产业政策 | 扶持政策=行业偏多 |
| 外盘影响 | 美股/恒指/A50期货 | 隔夜大涨=高开偏多 |
| 汇率 | 人民币/港币 | 本币升值=偏多 |

## 5. 趋势判断逻辑

### 5.1 评分体系

```
综合评分 = Σ(维度得分 × 维度权重)

维度得分范围: -3 ~ +3
  +3  强烈看多
  +2  偏多
  +1  弱多
   0  中性
  -1  弱空
  -2  偏空
  -3  强烈看空
```

### 5.2 默认权重

| 维度 | 权重 |
|---|---|
| 技术面 | 35% |
| 资金面 | 25% |
| 基本面 | 15% |
| 情绪面 | 15% |
| 宏观面 | 10% |

### 5.3 --focus 权重调整

| focus 值 | 调整后权重 |
|---|---|
| `technical` | 技术面 55%, 资金面 20%, 其他均分 |
| `capital_flow` | 资金面 50%, 技术面 20%, 其他均分 |
| `fundamental` | 基本面 45%, 宏观面 20%, 其他均分 |
| `sentiment` | 情绪面 45%, 技术面 25%, 其他均分 |

多个 focus 叠加时，权重平均分配后合并。

### 5.4 趋势判定

| 综合评分 | 趋势 |
|---|---|
| ≥ +2.0 | ▲ 看多 (Bullish) |
| ≤ -2.0 | ▼ 看空 (Bearish) |
| 其他 | ◆ 震荡 (Neutral) |

### 5.5 置信度

| 评分绝对值 | 置信度 |
|---|---|
| ≥ 2.5 | 高 |
| 2.0 ~ 2.5 | 中 |
| < 2.0 | 低（震荡区间） |

## 6. 输出格式

### 6.1 完整模式（默认）

见 `assets/report-template.md`，包含：
- 首行：分析日期（blockquote 样式）
- 标题行：股票名称、代码、周期
- 结论行：趋势方向、综合评分、置信度
- 关键信号表：五维得分与摘要
- 风险提示列表
- 操作建议区间（支撑/压力位）

### 6.2 精简模式（--compact）

```
{股票名称}({代码}) ▲看多 | 评分+2.1 | 技术+2 资金+1 基本0 情绪+1 宏观-1
风险: {关键风险1}; {关键风险2}
支撑/压力: {支撑位}/{压力位}
```

## 7. 特殊标的处理

| 标的类型 | 额外处理 |
|---|---|
| A股 ST/*ST | 基本面维度强制 -1；标题行标注退市风险 |
| 港股 | 增加恒指联动分析；标注卖空占比；无涨跌停限制提示 |
| ETF | 增加 IOPV 折溢价、跟踪误差、申赎清单分析 |
| 可转债 | 增加转股溢价率、纯债价值、强赎风险 |

## 8. 约束与免责

1. **非投资建议** — 所有输出仅供学习参考，不构成任何投资建议
2. **数据时效** — 分析基于最近交易日数据，存在延迟风险
3. **黑天鹅排除** — 无法预测突发事件（停牌、黑天鹅公告等）
4. **港股特殊** — 港股无涨跌停限制，波动风险需额外提示
5. **ETF特殊** — ETF 趋势同时参考 IOPV 净值与折溢价率

## 9. 错误处理

| 场景 | 处理 |
|---|---|
| 代码无法识别 | 提示格式要求并给出示例 |
| 标的已停牌 | 输出停牌提示，不生成趋势报告 |
| 数据不足 | 标注数据缺失维度，已有维度照常评分 |
| 多市场歧义 | 要求用户使用带后缀的代码明确指定 |
